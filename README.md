# Multiple-Pages
webpack 构建多页面应用

## 基础
- webpack, webpack-cli, webpack-dev-server
- css-loader, ts-loader, less-loader, sass-loader, url-loader, file-loader
- typescript

## 环境
- cross-env

## 对页面打包
- html-webpack-plugin
- mini-css-extract-plugin

## 代码压缩混淆
- terser-webpack-plugin

## Vue组件系统
- @babel/core, babel-loader
- @vue/babel-preset-jsx
- @vue/runtime-dom
